# レッスン復習アプリの仕組み

## 概要

Preplyのレッスン録音から、AIが自動的に以下を行います：

1. 音声を文字に変換する（文字起こし）
2. 先生の訂正・新しい単語を抽出する
3. 生徒が間違えたところを独自に分析する
4. Notionページにまとめる
5. フラッシュカードアプリに追加する

---

## ステップ 1: レッスンを録音する

Preplyにはレッスンの録音機能があります。レッスンが終わったら、録音ファイル（.mp3 / .m4a）をダウンロードします。

---

## ステップ 2: 音声を文字に変換する（文字起こし）

**ElevenLabs** という無料ツールを使います。

1. [elevenlabs.io](https://elevenlabs.io) にアクセスする
2. アカウントを作成する（Googleアカウントでログイン可能）
3. 左メニューの「Speech to Text」をクリック
4. 録音ファイルをアップロードする
5. 言語を「Japanese」に設定する
6. 「Transcribe」ボタンを押す
7. 数分待つと、文字起こし結果が表示される
8. 結果をダウンロードする（.txt または .srt 形式）

**ポイント：**
- ElevenLabsは話者を自動で分けてくれる（Speaker 0 / Speaker 1）
- 無料プランでも十分使える
- 精度はかなり高い（日本語の認識が優秀）

## ステップ 3: Claude Code で分析する

**Claude Code** はAnthropicが開発したAIコーディングツールです。ターミナル（コマンドライン）で動きます。

### インストール方法

```bash
npm install -g @anthropic-ai/claude-code
```

（Node.jsが必要です。[nodejs.org](https://nodejs.org) からインストールできます）

### 使い方

1. ターミナルを開く
2. `claude` と入力してEnterを押す
3. 文字起こしファイルのパスを伝える：

```
/path/to/transcript.txt - これは今日のレッスンの文字起こしです。
Speaker 0 は生徒、Speaker 1 は先生です。
分析して、言葉と文法をまとめてください。
```

4. Claude Code が自動的に：
   - 先生の訂正を抽出する
   - 新しい単語・フレーズをリストアップする
   - 生徒の発言を一文ずつチェックして、先生が見逃した間違いも見つける
   - すべてをNotionページに追加する
   - フラッシュカードアプリのデータを更新する

### スキル（自動化テンプレート）

Claude Code には「スキル」という機能があります。レッスン分析の手順をスキルとして保存しておくと、毎回同じ品質で分析してくれます。

スキルに含まれるルール：
- **言葉**：先生が教えたフレーズはそのまま保持する（単語だけ抜き出さない）
- **文法**：先生の訂正だけでなく、AIが独自に生徒の全発言をチェックする
- **目標**：1レッスンあたり15〜25個の文法項目を抽出する
- **説明は中国語**：生徒が中国語話者なので、文法の説明は中国語で書く

## ステップ 4: Notion に保存する

分析結果はNotionページに自動保存されます。

フォーマット：
```
▶ 四月二十三日
  ▶ 言葉
    - 量は質を凌駕する（りょうがする）：量凌驾于质
    - 暴露する（ばくろする）：揭露，曝光
  ▶ 文法
    - 風がすごく強かって
      → 風がすごく強くて
      - い形容词的て形：強い→強くて（不是強かって）
```

Claude Code はNotion APIを使って、直接ページを更新します。

## ステップ 5: フラッシュカードアプリ

### アプリの仕組み

- **データファイル**：`review_data.json` — 全ての単語と文法データが入っているJSONファイル
- **アプリ本体**：`index.html` — データを読み込んで表示するWebアプリ
- **ホスティング**：GitHub Pages で無料公開

### アプリの機能

- **単語カード**：日本語 → 中国語 / 中国語 → 日本語 の切り替え可能
- **文法チェック**：間違えた文 → 正しい文 → 解説
- **SRS（間隔反復）**：覚えた項目は出題間隔が長くなる

## 先生向けの活用アイデア

1. **レッスン後のフォローアップ**：生徒に分析結果を共有して、復習ポイントを明確にする
2. **進捗の可視化**：過去のレッスンデータを見れば、生徒の弱点パターンがわかる（例：助詞の間違いが多い、自他動詞の混同が繰り返される）
3. **スキルのカスタマイズ**：生徒のレベルや母語に合わせて、分析ルールを調整できる
4. **ホワイトボードの統合**：レッスン中のメモ（画像）も読み込んで、分析に統合できる
